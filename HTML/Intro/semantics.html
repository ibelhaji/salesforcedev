10.

< header >: Gebruiken voor introductietekst of logo. HTML-koppen bieden waardevolle informatie door belangrijke onderwerpen te benadrukken.

< footer >: Footers worden gebruikt voor het weergeven van een voettekst zoals bijv. copyright gegevens of bedrijfsinformatie.

< main >: Dit element vertegenwoordigt de dominante inhoud van een pagina; de belangrijkste informatie kan op deze manier worden weergegeven.

< nav >: Dit element vertegenwoordigt een sectie van een pagina die tot doel heeft navigatielinks te bieden, hetzij binnen het huidige document, hetzij naar andere documenten. Veelvoorkomende voorbeelden van navigatiesecties zijn menu's, inhoudsopgaven en indexen.

< section >: Dit element vertegenwoordigt een op zichzelf staand gedeelte van een document, dat geen specifieker semantics element heeft om het te vertegenwoordigen. Een voorbeeld is een speciale aanbieding op een pagina. Sections moeten altijd een header element hebben, op enkele uitzonderingen na.

< aside >: Dit element vertegenwoordigt een deel van een document waarvan de inhoud alleen indirect verband houdt met de hoofdinhoud van het document. Een voorbeeld zijn zijbalken van een webpagina.

< article >: Dit element vertegenwoordigt op zichzelf staande informatie in een document, pagina, toepassing of site, die bedoeld is om onafhankelijk verspreid of herbruikbaar te zijn. Voorbeelden zijn: een forumbericht, een krantenartikel of een blogbericht.
