Internet is een wereldwijd digitaal netwerk waarop computers kunnen worden aangesloten. www is een toepassing die gebruikt maakt van internet voor webpagina’s.

2.

HTML (Hyper Text Markup Language) is de standaard ‘opmaaktaal’ voor websites. HTML vormt steeds de basis en fundering van elke website. CSS (Cascading Style Sheets) bepaalt hoe de HTML van een webpagina wordt weergegeven. CSS omvat dus de opmaak en indeling van je website. Javascript is een programmeertaal voor het digitale web. Dankzij Javascript kunnen we HTML en CSS dynamisch maken. HTML, CSS en Javascript vormen samen de front-end van een website.

De Hummus-code is geen goede structuur. Er zijn geen koppen, geen alinea’s en er is geen ruimte tussen de tekst. Dit zorgt voor een onoverzichtelijke tekst.

7.

Een block HTML-element maakt altijd een nieuwe regel na de afsluitende tag, terwijl een inline HTML-element dat niet doet. Dus hoe de HTML ook wordt georganiseerd, block elementen creëren altijd een nieuwe regel. Met een inline-element is er nooit een nieuwe regel. Block-level elementen zijn meestal structurele elementen op de pagina. Inline-elementen bevinden zich in block elementen en omringen slechts kleine delen van de inhoud van het document.
